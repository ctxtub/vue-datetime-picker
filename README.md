# vue-datetime-picker

> VUE时间日期选择器

在线实例[DEMO](https://ctxtub.com/vue-datetime-picker/)

# 目录结构
```
|-- src
  |-- DateTimePickerDemo.vue // demo文件
  |-- main.js // demo文件入口
  |-- assets
  |   |-- stylus // 样式文件
  |       |-- base.styl
  |       |-- index.styl
  |       |-- mixin.styl
  |-- components // 时间日期选择器组件
      |-- DateTimePicker.vue
      |-- index.js
```

## NPM命令
```
npm install
npm run serve
npm run build
npm run build:lib // 组件打包
```
